# Проект: Напоминатель ЭЦП для сотрудников ГСН

## Цель
Мобильное приложение + Telegram бот для отслеживания сроков окончания ЭЦП сотрудников службы государственного строительного надзора.

## Платформа
- **Mobile**: Flutter (Android + iOS)
- **Bot**: Python + Telegram Bot API
- **База**: SQLite

## Текущий статус
- [ ] Этап 0: Подготовка (создание бота, установка Flutter)
- [ ] Этап 1: Telegram бот
- [ ] Этап 2: Flutter приложение
- [ ] Этап 3: Доработка

## Принятые решения
- Telegram выбран как канал уведомлений
- Локальная SQLite для хранения данных
- Планировщик проверяет сроки ежедневно
